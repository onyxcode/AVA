An attempt to bring together a ton of API's into an easy-to-use central hub.

## Install and run

Clone the repo to your computer.

```
git clone https://github.com/onyxcode/AIO-API/tree/v-rewrite.git -b v-rewrite
```

Start the API.

```
v run api.v
```
