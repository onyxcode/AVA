# _**A**(ll-in-one)**V**(lang)**A**(PI)_


An attempt to bring together a ton of API's into an easy-to-use central hub.

## Install and run

Clone the repo to your computer.

```
git clone https://github.com/onyxcode/AVA.git
```

CD into the directory.

```
cd AVA
```

Start the API.

```
v run .
```
